# FastAPI
Repositorio con tutoriales y ejemplos de cómo usar FastAPI
## Ejecución
Para ejecutar FastAPI tiene que instalarse `uvicorn` y ejecutar en la terminal el comando:
`uvicorn myapi:app --reload`
